# ShoeHub

**ShoeHub** is an online shoe store built with **Django** that lets users browse shoes by category, view details, add items to a shopping cart, and manage their cart. The site features a **landing page**, **shimmering background**, and a **responsive design**.

## Features

- Landing page with **Get Started** button.
- Browse shoes by categories: Men, Women, Kids, Sports, Casual, Formal, Boots.
- **Shoe detail page** with “Add to Cart” and **Back to Home** buttons.
- Shopping cart: add items, increase/decrease quantities, clear cart.
- About Us and Help Desk pages with **light background**.
- Responsive navbar: Home, About, Contact, Help Desk.
- Modern UI with **warm shadows**, **glowing buttons**, and **shimmering gradient background**.
